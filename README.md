# FIR Low-Pass Filtering for Noisy Measurement Data

This project was completed for the Engineering Experimentation and Data Analysis course. The main objective of the project is to apply FIR low-pass filtering to noisy measurement data and compare the filtered signal with the original noise-free reference signal.

The dataset includes a time vector, a noisy measurement signal, and an original reference signal. First, the measured and original signals were compared in the time domain. Then, FFT-based frequency-domain analysis was performed to identify the dominant frequency components and understand the noise characteristics of the signal.

A Finite Impulse Response (FIR) low-pass filter was designed by selecting suitable filter parameters such as cut-off frequency, transition width, and filter order. The filter was applied using MATLAB, and the filtered signal was compared with the original signal to evaluate the filtering performance.

## Key Topics

* Signal processing
* Sampling frequency
* Nyquist theorem and aliasing
* FFT and frequency-domain analysis
* FIR low-pass filter design
* Cut-off frequency selection
* Filter order and transition width
* Noise reduction
* MATLAB data analysis
